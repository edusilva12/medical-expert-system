## Proyecto en Ionic

### Instalar dependencias del repositorio
```bash
npm install
```
### Ejecutar proyecto
```bash
ionic serve
```
### Construir proyecto
```bash

```
## Android
### Crear proyecto android
```bash
ionic cap add android
```
### Sincronizar cambios del proyecto
```bash
ionic cap sync android
```
### Abrir Android Studio
```bash
ionic cap open android
```
